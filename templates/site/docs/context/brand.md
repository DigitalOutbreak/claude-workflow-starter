## Tone callouts for AI agents

When you're writing or rewriting copy on this site:

- Start with the noun, not the brand. The reader is the subject.
- One idea per sentence. If you used "and" twice, split it.
- Specifics beat adjectives. "47 launches" beats "many launches."
- Cut the throat-clearing first sentence. The second sentence is usually the real start.
- If a sentence could appear on any agency's site, rewrite it or delete it.
